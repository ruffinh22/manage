Notes

Les notes sont importées depuis un fichier CSV.

Étudiants

Les étudiants doivent être créés avant l'importation des notes, si un étudiant n'est pas crée, ses notes seront ignorées.

Les informations sur les étudiants peuvent être complétées une fois l'importation terminée

Pour faire un affichage complet des notes d'un étudiant,cet étudiant doit avoir des notes de saisi

Matières

Les matières doivent être créées avant l'importation des notes, si une matière n'est pas créée, les notes seront ignorées.

Les matières peuvent être liées manuellement a des UE

UE

Les UE doivent être créées et rattachées manuellement à une ou plusieurs matières après importation
